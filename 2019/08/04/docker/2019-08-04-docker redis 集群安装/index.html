cluster-announce-ip 为宿主机服务器IP

创建redis配置文件

在redis-cluster目录下创建并执行批处理命令，用于创建redis配置文件。
创建端口号为从7000 到 7005 的配置文件
for port in seq 7000 7005; do
mkdir -p ./${port}/conf
&& PORT=${port} envsubst < ./redis-cluster.tmpl > ./${port}/conf/redis.conf
&& mkdir -p ./${port}/data;
done
